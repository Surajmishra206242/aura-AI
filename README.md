Aura AI Chat is a modern web-based conversational AI application that allows users to interact with an intelligent chatbot in real time. Built with a clean and responsive UI, the project provides a smooth chat experience similar to popular AI assistants.

The application is developed using Node.js and Express.js for backend processing and routing, while the frontend is built using HTML, CSS, and JavaScript with Bootstrap for styling. It supports dynamic messaging, session-based interactions, and a structured chat interface.

This project demonstrates full-stack development skills, including API handling, frontend design, and backend integration.

🚀 Key Features:
Real-time chat interface
Responsive UI (mobile + desktop)
Clean and modern design
Backend powered by Node.js & Express
Easy to extend with AI APIs
🛠️ Tech Stack:

HTML, CSS, JavaScript, Bootstrap, Node.js, Express.js
